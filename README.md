📊 Amazon Sales Analysis Dashboard (Power BI)
📌 Project Overview

This project analyzes Amazon sales data to extract actionable business insights using Power BI.

🎯 Objectives

Analyze overall sales performance

Identify top-performing categories and sizes

Understand fulfillment efficiency

Explore geographical sales distribution

Segment B2B vs B2C customers

📈 Dashboard Structure
Page 1 – Executive Overview

Total Sales

Total Orders

Average Order Value

Monthly Sales Trend

Sales by Category

Sales by State

Page 2 – Product & Customer Analysis

Quantity by Size

Top Cities by Sales

B2B vs B2C Analysis

Page 3 – Operations & Geography

Fulfilment Performance

Service Level Analysis

Geographic Sales Map

💡 Key Insights

T-Shirts generate highest revenue

Medium and Large sizes dominate sales

Majority sales from Maharashtra and Karnataka

99% revenue from B2C segment

Amazon Fulfilment handles majority orders
